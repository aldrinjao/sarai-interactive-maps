EVI Maps

The enhanced vegetation index (EVI) is an 'optimized' vegetation index designed to enhance the vegetation signal with improved sensitivity in high biomass regions and improved vegetation monitoring. Identify the EVI date coverage to get started. Click on any point on the map to view more information.

Project 2.1 Community-Level SARAI-Enhanced Agricultural Monitoring System (CL SEAMS)

Project 3.1 Knowledge Management and Mobile Applications for Agriculture

How to use

  1. Select a starting date.

  2. Select an end date.

  3. Select a province.

  4. Click Filter.

  5. Map Source - Select one from the many types of maps to have different views of the selected location.

  1. Zoom in and out of the map to have a detailed view.

  2. Measure the area of a specific location using this tool.

  3. Drag the slider to make the slider lighter or darker.

  4. Click on any point on the map to view the coordinates, EVI Times Series Chart, and NDVI Day of the Year Chart
