NDVI Maps

Using available satellite data, SARAi NDVI map can monitor the flood and damage extent during typhoons which can be used during field assessments. Through this monitoring scheme, affected communities can easily be located. SARAi NDVI map can also provide a near real-time estimation of the planted areas nationwide or per region. Greener color indicates existence of vegetation.

How to use

  1. Select a starting date.

  2. Select an end date.

  3. Select a province.

  4. Click Filter.

  5. Map Source - Select one from the many types of maps to have different views of the selected location.

  1. Zoom in and out of the map to have a detailed view.

  2. Measure the area of a specific location using this tool.

  3. Drag the slider to make the slider lighter or darker.

  4. Click on any point on the map to view the coordinates, NDVI Times Series Chart, and NDVI Day of the Year Chart
